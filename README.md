# Snake Game

A bright, animated Snake game built with Pygame. It includes a stylized yellow theme, animated snake and food, multiple difficulty modes, and power-ups.

## Features
- Bright yellow visual theme with animated background
- Realistic snake details (eyes, tongue, scales)
- Smooth food animation with glow and orbiting sparkle
- Difficulty modes: Easy, Medium, Hard
- Power-ups: slow-time, magnet, shield, double points
- Pause and menu controls
- Resizable window

## Controls
- Arrow Keys: Move
- Mouse: Use on-screen buttons (Pause, Resume, Menu)

## Power-ups
- `slow`: slows the game speed for 6 seconds
- `magnet`: pulls food toward the snake when nearby
- `shield`: absorbs one collision
- `double`: doubles points per food

## How to Run
```bash
py -3.13 "d:/My projects/Snake game/snake_game.py"
```

## Requirements
- Python 3.13+
- Pygame 2.6.1+

## Notes
- High scores are stored in `high_score.txt`.
